# 🤖 AI Supplier Risk Copilot

An intelligent MVP for SMEs in import/export to track suppliers,
monitor shipments, and get predictive risk alerts.

## 🏗️ Tech Stack

| Layer      | Technology                          |
|------------|-------------------------------------|
| Frontend   | Next.js + TypeScript + Tailwind CSS |
| Backend    | FastAPI (Python)                    |
| Database   | PostgreSQL                          |
| AI Module  | scikit-learn (MVP)                  |
| DevOps     | Docker + docker-compose             |

## 🚀 Getting Started
```bash
cp .env.example .env
docker-compose up --build
```

## 🌐 Access
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000/docs
